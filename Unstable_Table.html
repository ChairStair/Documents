Unstable Table


Unstable Table is a S before I opener, and S before L/Z


1st Bag // T-Spin Mini


The first bag requires S before I.

The alternate build requires L before S, and J before O before I.


2nd Bag // Cut Copy


The L O J is interchangeable.

It is recommended to freestyle after this.

To tuck the I piece,

Soft Drop -> CW(While holding down soft drop) -> CW


3rd Bag // T-spin Double

The L O J is interchangeable.

Hold the I piece.

This build will fail with I > Z, L S J > O


4th Bag // T-Spin Double


The J and L are interchangeable.


5th Bag // STSD


Fill in the yellow square for a overhang to make a STSD
